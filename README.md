# 🛒 E-Commerce Cart Application

A full-stack *E-Commerce Cart Application* built using *React.js, Node.js, Express.js, and Neon PostgreSQL*.  
The application allows users to browse products, manage a shopping cart, and interact with a scalable backend API connected to a cloud database.  
Deployed using *Render (Frontend + Backend Hosting)*.

---

## 🚀 Live Demo

- 🌐 Frontend: https://codealpha-simplee-commerce-store-1.onrender.com/
- ⚙️ Backend API: "https://codealpha-simplee-commerce-store-jesr.onrender.com"; 

---

## 📌 Features

- 🛍️ Product listing and browsing
- 🔍 Search products
- ➕ Add products to cart
- ➖ Remove products from cart
- 🔄 Update product quantity
- 📦 REST API integration
- 💾 Neon PostgreSQL database integration
- 📱 Fully responsive UI (Mobile + Desktop)
